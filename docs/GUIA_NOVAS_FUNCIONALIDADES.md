# 🎮 Guia Rápido - Novas Funcionalidades

## 🎯 Quiz de Vocabulário

### Como Usar:
1. Abra o dashboard (clique no ícone da extensão)
2. Clique na aba **"Quiz"**
3. Leia a palavra em inglês
4. Escolha a tradução correta entre 4 opções
5. Veja o resultado instantâneo (verde = correto, vermelho = errado)
6. Ao final, veja sua pontuação e clique "Jogar Novamente"

### Requisitos:
- Mínimo de 4 palavras salvas no deck

---

## 🔍 Filtro de Plataforma

### Como Usar:
1. Dashboard → Aba **"Vocabulário"**
2. Localize o dropdown **"Todas plataformas"** no topo
3. Selecione: YouTube, Netflix, HBO Max, etc
4. A tabela filtra automaticamente

### Funcionalidades:
- Mostra apenas palavras da plataforma selecionada
- Mantém seleção ao trocar de aba
- Combina com outros filtros (busca, status, tags)

---

## 📊 Heatmap de Atividade

### Como Usar:
1. Dashboard → Aba **"Estatísticas"**
2. Veja o grid de 90 dias no topo
3. Passe o mouse sobre um quadrado para ver detalhes
4. Cores indicam intensidade:
   - Cinza escuro = 0 palavras
   - Azul claro = poucas palavras
   - Azul médio = algumas palavras
   - Azul forte = muitas palavras
   - Azul intenso = MUITAS palavras

### Interpretação:
- Identifique seus dias mais produtivos
- Veja padrões de estudo
- Mantenha uma sequência (streak)

---

## 📈 Gráfico de Progresso Semanal

### Como Usar:
1. Dashboard → Aba **"Estatísticas"**
2. Role para baixo até o gráfico de barras
3. Veja as últimas 12 semanas
4. Passe o mouse sobre uma barra para ver detalhes

### Interpretação:
- Barras altas = semanas produtivas
- Identifique tendências de crescimento
- Compare semanas diferentes

---

## 🔄 Sincronização Tempo Real

### Como Funciona:
1. Assista um vídeo com LinguaFlow ativo
2. Clique em uma palavra para salvar
3. O dashboard atualiza **automaticamente**
4. Não precisa dar refresh!

### Indicadores:
- Contador de palavras aumenta instantaneamente
- Notificação visual no dashboard
- Badge do ícone da extensão atualiza

---

## 💾 Backup Automático

### Como Usar:
1. Dashboard → Aba **"Configurações"**
2. Role até **"Backup Automático"**
3. Veja status do último backup
4. Clique **"Fazer Backup Agora"** para backup manual
5. Clique **"Restaurar Backup"** para restaurar

### Funcionalidades:
- Backup automático a cada 24 horas
- Salvo localmente no navegador
- Inclui todas as palavras e progresso
- Restauração com um clique

---

## 🎨 Palavras Coloridas nas Legendas

### Como Funciona:
- Palavras aparecem coloridas automaticamente
- Cores indicam seu nível de conhecimento:

### Legenda de Cores:
- 🟢 **Verde claro** = Já conheço (marcada como conhecida)
- 🟢 **Verde** = Dominada (mature - intervalo > 21 dias)
- 🔵 **Azul** = Revisando (review - estudando regularmente)
- 🟡 **Amarelo** = Aprendendo (learning - recém-adicionada)
- 🔵 **Azul claro** = Salva no deck (new - nunca revisada)
- ⚪ **Branco** = Nova (nunca vista antes)

### Como Usar:
1. Assista qualquer vídeo
2. Palavras aparecem coloridas automaticamente
3. Clique em uma palavra para ver detalhes
4. Salve palavras novas (brancas) para começar a estudá-las

## 📚 Estatísticas Avançadas

### Como Usar:
1. Dashboard → Aba **"Vocabulário"**
2. Clique na sub-aba **"Words"**
3. Veja estatísticas no topo:
   - Total de palavras únicas
   - Cobertura Top-5k
   - Palavras por categoria (nova, salva, aprendendo, etc)

### Bandas de Frequência:
- **1-100**: Palavras mais comuns (ouro)
- **101-1000**: Muito comuns (laranja)
- **1001-3000**: Comuns (azul)
- **3001-5000**: Menos comuns (cinza)
- **Fora do Top-5k**: Raras

### Como Interpretar:
- Alta cobertura Top-5k = bom vocabulário básico
- Muitas palavras "novas" = oportunidade de aprendizado
- Palavras "dominadas" = progresso consolidado

## ❓ Perguntas Frequentes

**P: O Quiz não aparece?**
R: Você precisa ter pelo menos 4 palavras salvas no deck.

**P: O Heatmap está vazio?**
R: Comece salvando palavras! O heatmap mostra atividade dos últimos 90 dias.

**P: As cores não aparecem nas legendas?**
R: Certifique-se de que a extensão está ativa e você salvou algumas palavras.

**P: O backup automático funciona offline?**
R: Sim! O backup é salvo localmente no navegador.

**P: Posso exportar meu progresso?**
R: Sim! Use "Backup Completo" nas configurações para exportar um arquivo JSON.
