## 🎯 Visão Geral

O Motor de Análise Linguística do LinguaFlow agora suporta **QUALQUER IDIOMA** de forma dinâmica. A IA detecta automaticamente o idioma da legenda e mantém todas as análises nesse idioma, explicando em português.

## 🧪 Exemplos por Idioma

### 1. Inglês (English)

**Input:**
```
Palavra: "get over"
Contexto: "She got over her fear"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   ⚡ PHRASAL VERB

2. 📌 Significado Contextual
   "Get over" significa "superar" ou "recuperar-se de" algo negativo.

3. 📚 Estrutura e Uso
   - Padrão: [subject] + get over + [object]
   - Colocações: "get over fear", "get over illness", "get over someone"

4. 🔄 Sinônimos e Nuances
   1. "overcome" - Mais formal
   2. "recover from" - Foco em recuperação
   3. "move past" - Informal

5. 💡 Exemplos de Uso
   1. "I can't get over how beautiful this is" → Não consigo acreditar
   2. "Get over it!" → Supera!
```

---

### 2. Espanhol (Español)

**Input:**
```
Palavra: "echar de menos"
Contexto: "Te echo de menos"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   Expressão idiomática (locución verbal)

2. 📌 Significado Contextual
   "Echar de menos" significa "sentir falta de" ou "ter saudades de" alguém.

3. 📚 Estrutura e Uso
   - Padrão: [sujeto] + echar de menos + [objeto]
   - Colocações: "echar de menos a alguien", "echar de menos algo"

4. 🔄 Sinônimos e Nuances
   1. "extrañar" - Mais comum na América Latina
   2. "añorar" - Mais formal, literário
   3. "tener nostalgia de" - Muito formal

5. 💡 Exemplos de Uso
   1. "Echo de menos mi casa" → Sinto falta da minha casa
   2. "¿Me echas de menos?" → Você sente minha falta?
```

---

### 3. Francês (Français)

**Input:**
```
Palavra: "avoir envie de"
Contexto: "J'ai envie de partir"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   Locução verbal (expression verbale)

2. 📌 Significado Contextual
   "Avoir envie de" significa "ter vontade de" ou "estar com vontade de".

3. 📚 Estrutura e Uso
   - Padrão: [sujet] + avoir envie de + [infinitif/nom]
   - Colocações: "avoir envie de faire", "avoir envie d'un café"

4. 🔄 Sinônimos e Nuances
   1. "vouloir" - Mais direto, menos emocional
   2. "désirer" - Mais formal, desejo forte
   3. "avoir le goût de" - Regional (Québec)

5. 💡 Exemplos de Uso
   1. "J'ai envie de dormir" → Estou com vontade de dormir
   2. "Tu as envie d'un café?" → Você quer um café?
```

---

### 4. Alemão (Deutsch)

**Input:**
```
Palavra: "aufhören"
Contexto: "Hör auf damit!"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   Verbo separável (trennbares Verb)

2. 📌 Significado Contextual
   "Aufhören" significa "parar" ou "cessar". No imperativo "Hör auf!" = "Pare!"

3. 📚 Estrutura e Uso
   - Padrão: [Subjekt] + hören + ... + auf
   - Colocações: "aufhören zu arbeiten", "aufhören mit etwas"

4. 🔄 Sinônimos e Nuances
   1. "stoppen" - Mais abrupto, parada súbita
   2. "beenden" - Terminar completamente
   3. "einstellen" - Cessar atividade

5. 💡 Exemplos de Uso
   1. "Ich höre auf zu rauchen" → Vou parar de fumar
   2. "Hör auf damit!" → Pare com isso!
```

---

### 5. Japonês (日本語)

**Input:**
```
Palavra: "頑張る" (ganbaru)
Contexto: "頑張ってください"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   Verbo godan (五段動詞)

2. 📌 Significado Contextual
   "頑張る" (ganbaru) significa "se esforçar", "fazer o melhor", "persistir".

3. 📚 Estrutura e Uso
   - Padrão: [主語] + 頑張る
   - Colocações: "頑張ってください" (ganbare kudasai), "頑張ります" (ganbarimasu)

4. 🔄 Sinônimos e Nuances
   1. "努力する" (doryoku suru) - Mais formal, "fazer esforço"
   2. "励む" (hagemu) - Dedicar-se com afinco
   3. "精を出す" (sei wo dasu) - Colocar energia

5. 💡 Exemplos de Uso
   1. "試験、頑張ってね" → Boa sorte na prova (lit: se esforce)
   2. "毎日頑張っています" → Me esforço todos os dias
```

---

### 6. Chinês (中文)

**Input:**
```
Palavra: "加油" (jiāyóu)
Contexto: "加油！你可以的！"
```

**Output:**
```
1. 🎯 Tipo de Expressão
   Expressão idiomática (成语 chéngyǔ)

2. 📌 Significado Contextual
   "加油" (jiāyóu) literalmente "adicionar óleo", significa "força!", "vai!", "você consegue!".

3. 📚 Estrutura e Uso
   - Padrão: 加油 (standalone ou antes de frase)
   - Colocações: "加油加油" (repetido para ênfase), "一起加油" (vamos juntos)

4. 🔄 Sinônimos e Nuances
   1. "努力" (nǔlì) - Mais formal, "esforçar-se"
   2. "坚持" (jiānchí) - "Persistir", "continuar"
   3. "加把劲" (jiā bǎ jìn) - "Dar um gás extra"

5. 💡 Exemplos de Uso
   1. "考试加油！" → Boa sorte na prova!
   2. "我们一起加油吧" → Vamos juntos, força!
```


## 📊 Detecção Automática de Idioma

### Como a IA Detecta

A IA usa o **contexto da legenda** para detectar automaticamente o idioma:

1. **Análise de caracteres** - Detecta alfabeto (latino, cirílico, CJK, árabe)
2. **Padrões linguísticos** - Identifica estruturas gramaticais típicas
3. **Palavras-chave** - Reconhece palavras comuns de cada idioma
4. **Contexto** - Usa a frase completa para confirmar

### Exemplos de Detecção

```javascript
// Inglês
"I see the guys on D" → Detecta: artigos (the), pronomes (I), estrutura SVO

// Espanhol
"Te echo de menos" → Detecta: pronome átono (te), verbo (echo), preposição (de)

// Japonês
"頑張ってください" → Detecta: caracteres kanji/hiragana, partícula (て)

// Chinês
"加油！你可以的！" → Detecta: caracteres hanzi, partícula (的)

// Árabe
"أنا أحبك" → Detecta: escrita da direita para esquerda, caracteres árabes
```

## ⚠️ Casos Especiais

### Idiomas com Múltiplas Variantes

#### Inglês
- 🇺🇸 Americano: "color", "favorite", "gotten"
- 🇬🇧 Britânico: "colour", "favourite", "got"
- 🇦🇺 Australiano: gírias específicas ("mate", "arvo")

**Solução:** IA detecta e menciona a variante quando relevante.

#### Espanhol
- 🇪🇸 Espanha: "vosotros", "ordenador", "coche"
- 🇲🇽 México: "ustedes", "computadora", "carro"
- 🇦🇷 Argentina: "vos", "computadora", "auto"

**Solução:** IA identifica regionalismos e explica diferenças.

#### Chinês
- 🇨🇳 Simplificado: 汉字
- 🇹🇼 Tradicional: 漢字

**Solução:** IA reconhece ambos e mantém o original.
